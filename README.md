# Opis rozdziałów

Ten zestaw plików RST zawiera trzy rozdziały dotyczące rozwoju sztucznej inteligencji:

### 1. Wprowadzenie
Podstawowe informacje o historii i genezie sztucznej inteligencji, jej dynamicznym wzroście oraz znaczeniu we współczesnym świecie.

### 2. Technologie AI
Opis kluczowych technologii wykorzystywanych w sztucznej inteligencji. Ten rozdział zawiera również przykładowy obrazek sieci neuronowej.

### 3. Zastosowania AI
Przegląd praktycznych zastosowań sztucznej inteligencji w różnych sektorach, wzbogacony o tabelę przedstawiającą przykłady użycia.

Pliki są gotowe do użycia w dokumentacji Sphinx.
